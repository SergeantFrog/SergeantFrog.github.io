RFID Timing Software

(Project code is not publically uploaded)

This was my senior design project for the University of Connecticut. We used RFID chips to read the timing of racers by reading radio frequencies, which were then uploaded to our database.
My involvement in this project was to read the bytes from the frequencies and manage the database. I then created automated SQL statements to push out information into our created tables. These tables then connected to the created website and were pushed out so that they could be visualized.
